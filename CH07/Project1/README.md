### Flow of the Program

Display Menu 
1. Convert from C to F
2. Convert from F to C
3. Exit

If User Selects 1:
while(user has values to convert):
    ReadAndValidateCharConversion()
    ConvertCToF()
    DisplayResult()
    OfferToGoBackOrConvert()

if user selects 2:
while(user has values to convert):
    ReadAndValidateCharConversion()
    ConvertFToC()
    DisplayResult()
    OfferToGoBackOrConvert()

if user selects 3: 
exit the Program.
